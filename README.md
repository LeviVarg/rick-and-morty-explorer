# Rick & Morty Explorer

A single-page application built with Next.js, TypeScript, and Tailwind CSS that lets you browse and search Rick & Morty characters using the [Rick and Morty API](https://rickandmortyapi.com/).

## Getting Started

First, run the development server:

```bash
npm i
#then to run th dev server
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev

or build the project

npm run build

npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
